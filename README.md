# JobSearcher 2.0

An AI-powered career assistant that helps candidates find jobs that truly match their experience.

## Features

- Resume upload
- Resume parsing
- Skill extraction
- Contact extraction
- Candidate profile generation
- Adzuna job search
- Visa eligibility filtering
- Semantic job matching
- Resume optimization
- Application tracking

## Tech Stack

Backend
- Python
- FastAPI

NLP
- spaCy
- Sentence Transformers
- RapidFuzz

Database
- SQLite (development)

Future Frontend
- React
